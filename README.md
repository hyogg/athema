# athema
멀티캠퍼스 웹서비스 백엔드 개발 4회차 팀 프로젝트 by 3조 포비와 친구들

## 1. 프로젝트 주제
- ‘어떠한’ 여행을 떠나고 싶은지 먼저 묻고, 그에 따라서 맞춤 여행 코스 추천 및 예약 진행
- 여행의 목적에 맞게 계획을 짜기 어려워하거나, 정보 수집이 어렵게 느껴지는 사람에게 유용한 서비스
- Athema = '어때?' + '테마'
